### Descrição do Jogo

FitGame é um jogo de plataforma vertical infinito, desenvolvido em Phaser 3 e TypeScript.

### Regras do Jogo
1. O jogador começa com 3 vidas.
2. Tocar em um halter reduz 1 vida.
4. Cada whey apanhado vale 100 pontos.

### Jogabilidade e Controlos

Use as setas esquerda e direita para controlar a direção do nosso maromba.

### Como executar

Abra o terminal na pasta raiz do projeto e digite:
npm install
npm run dev
